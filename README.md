## Описание проекта

Решение задачи прогнозирования оттока клиентов бизнес-сегмента Альфа-Банка. Проект направлен на раннее выявление клиентов с высоким риском оттока и определение ключевых факторов, влияющих на их уход.

**Цель:** Построить ML-модель для предсказания оттока клиентов и разработать рекомендации по стратегии удержания.

**Бизнес-контекст:**
- Глобальный уровень оттока: 7.1%
- "Зона смерти": 3-12 месяц жизни клиента в банке (отток до 18%)
- Экономия на удержании клиента дешевле привлечения нового

## Ключевые результаты

- **ROC-AUC модели: 0.9012** - высокая прогнозная способность
- **Выявлены 5 сегментов оттока** с уникальными характеристиками
- **Определены ключевые драйверы оттока** через SHAP-анализ
- **Разработаны конкретные бизнес-рекомендации** для снижения оттока
